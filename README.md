# Rain Alert SMS Notification App ☔

This Python project uses the OpenWeather API and Twilio API to detect upcoming rain conditions and automatically send an SMS alert reminder to carry an umbrella. The app fetches weather forecast data, checks weather condition codes, and sends real-time notifications using Twilio messaging services.

## Features
- Fetches weather forecast using OpenWeather API
- Detects rain conditions automatically
- Sends SMS alerts using Twilio
- Simple and beginner-friendly Python project

## Technologies Used
- Python
- Requests
- Twilio API
- OpenWeather API

## Installation

```bash
pip install requests twilio
```

## Run the Project

```bash
python main.py
```
